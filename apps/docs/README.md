# DeonPay API Documentation

Official documentation site for the DeonPay API.

## Features

- **Migration Guide**: Step-by-step guide for migrating from Stripe or custom implementations
- **Release Notes**: Version history, breaking changes, and migration paths
- **API Reference**: Complete OpenAPI 3.1 specification with all endpoints

## Running Locally

```bash
npm install
npm run dev
```

Visit `http://localhost:3004` to view the documentation.

## Building for Production

```bash
npm run build
npm start
```

## Deployment

This site is deployed on Vercel at `https://docs.deonpay.mx`

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- React 19
